# Getting started

Navigate into project directory

```sh
cd unixCopier
```

Run below command to do the copying while removing duplicates

```sh
sudo chmod u+x copy.sh && ./copy.sh
```

## Testing

To test that above is working run:

```sh
bash test_script.sh
```

Enjoy!
